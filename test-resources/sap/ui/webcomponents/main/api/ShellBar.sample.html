Overview

The ui5-shellbar is meant to serve as an application header and includes numerous built-in features, such as: logo, profile icon, title, search field, notifications and so on.

ES6 Module Import

import "@ui5/webcomponents/dist/ShellBar";

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description
logo
String
Defines the logo source URI.
notificationCount
notification-count
String
Defines the notificationCount, displayed in the notification icon top-right corner.
primaryTitle
primary-title
String
Defines the primaryTitle.
profile
URI
Defines URI of the profile action. If no URI is set - profile will be excluded from actions.
secondaryTitle
secondary-title
String
Defines the secondaryTitle.

Note: On smaller screen width, the secondaryTitle would be hidden.
showCoPilot
show-co-pilot
boolean
Defines, if the product CoPilot icon would be displayed.
showNotifications
show-notifications
boolean
Defines, if the notification icon would be displayed.
showProductSwitch
show-product-switch
boolean
Defines, if the product switch icon would be displayed.

Children

This Web Component accepts other HTML Elements as children. Use the data-ui5-slot attribute to define the category of each child, if more than one category is accepted. You can provide multiple children for the categories marked with [0..n] or just one otherwise.

Category
Type
Description
icon
HTMLElement [0..n]
Defines a ui5-icon in the bar that will be placed in the beginning.
items
HTMLElement
Defines the ui5-shellbar aditional items.

Note: You can use the  <ui5-shellbar-item></ui5-shellbar-item>.
searchField
HTMLElement [0..n]
Defines the ui5-input, that will be used as a search field.

ShellBarItem

The ui5-shellbar-item represents a custom item, that might be added to the ui5-shellbar.

ES6 Module Import

import "@ui5/webcomponents/dist/ShellBarItem";

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description

URI
Defines the item source URI.

Description
