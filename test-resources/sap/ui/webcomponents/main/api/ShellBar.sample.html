Overview

The ui5-shellbar is meant to serve as an application header and includes numerous built-in features, such as: logo, profile icon, title, search field, notifications and so on.

ES6 Module Import

import "@ui5/webcomponents/dist/ShellBar";

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description
logo
string
Defines the logo source URI.
notificationCount
notification-count
string
Defines the notificationCount, displayed in the notification icon top-right corner.

primary-title
string
Defines the primaryTitle.

string
Defines the source URI of the profile action. If no source is set - profile will be excluded from actions.
secondaryTitle
secondary-title
string
Defines the secondaryTitle.

Note: On smaller screen width, the secondaryTitle would be hidden.

boolean

Defines, if the product CoPilot icon would be displayed.
showNotifications

boolean
Defines, if the notification icon would be displayed.

show-product-switch
boolean

Defines, if the product switch icon would be displayed.

Children

This Web Component accepts other HTML Elements as children. Use the slot attribute to define the category of each child, if more than one category is accepted.

Category
Type
Description
default
HTMLElement [0..n]
Defines the ui5-shellbar aditional items.

Note: You can use the  <ui5-shellbar-item></ui5-shellbar-item>.
icon
HTMLElement [0..n]
Defines a ui5-icon in the bar that will be placed in the beginning.
menuItems
HTMLElement [0..n]
Defines the items displayed in menu after a click on the primary title.

Note: You can use the  <ui5-li></ui5-li> and its ancestors.
searchField
HTMLElement [0..n]
Defines the ui5-input, that will be used as a search field.
