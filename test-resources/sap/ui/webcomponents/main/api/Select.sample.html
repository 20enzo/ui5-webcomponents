Overview

The ui5-select component is used to create a drop-down list. The items inside the ui5-select define the available options by using the ui5-li component.

Keyboard Handling

The ui5-select provides advanced keyboard handling. If the ui5-select is focused, you can open or close the drop-down by pressing F4, ALT+UP or ALT+DOWN keys. Once the drop-down is opened, you can use the UP and DOWN arrow keys to navigate through the available options and select one by pressing the Space or Enter keys.

ES6 Module Import

import "@ui5/webcomponents/dist/Select";
import "@ui5/webcomponents/dist/StandardListItem"; (ui5-li)

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description
disabled
boolean
Defines whether ui5-select is in disabled state.

Note: A disabled ui5-select is noninteractive.
valueState
value-state
string
Defines the value state of ui5-select. Available options are: None, Success, Warning and Error.

Children

This Web Component accepts other HTML Elements as children. Use the data-ui5-slot attribute to define the category of each child, if more than one category is accepted. You can provide multiple children for the categories marked with [0..n] or just one otherwise.

Category
Type
Description
items
HTMLElement [0..n]
Defines the ui5-select items.

Note: Only one selected item is allowed. If more than one item is defined as selected, the last one would be considered as the selected one.

Note: Use the ui5-li component to define the desired options.
