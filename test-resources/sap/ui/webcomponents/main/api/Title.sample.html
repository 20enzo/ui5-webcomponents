Overview

The ui5-title component is used to display titles inside a page. It is a simple, large-sized text with explicit header/title semantics.

ES6 Module Import

import "@ui5/webcomponents/dist/Title";

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description
level
String
Defines the title level. Supported values are from H5 to H1.
wrap
Boolean
false
Determines whether the ui5-title should wrap.

Children

The content of this Web Component is interpreted as text and shown to the user. All HTML Elements inside the Web Component are ignored - only their text is used.
