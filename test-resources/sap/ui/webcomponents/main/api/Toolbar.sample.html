Overview

The ui5-toolbar component is a horizontal container, most commonly used to display buttons, labels, selects, and various other input components.

Usage

You can add horizontal space between the ui5-toolbar items, using the ui5-toolbarspacer. For example, it can be useful if you want to push an item to the edge of the ui5-toolbar.

ES6 Module Import

import "@ui5/webcomponents/dist/Toolbar";
Note: This also includes the ui5-toolbar-spacer Web Component.

Properties/Attributes

You can use both properties and attributes with the same effect. The name of each attribute is listed below the name of the property, if different.

Name
Type
Default Value
Description
design
ToolbarDesign
Defines the ui5-toolbar design. Available options are: Solid, Transparent, and Info.

Note: Design settings are theme dependent. They also determine the default height of the ui5-toolbar.
toolbarStyle
toolbar-style
ToolbarStyle
Defines the visual style of the ui5-toolbar. Available options are: Standard and Clear.

Note: The visual styles are theme dependent.

Children

This Web Component accepts other HTML Elements as children. Use the data-ui5-slot attribute to define the category of each child, if more than one category is accepted. You can provide multiple children for the categories marked with [0..n] or just one otherwise.
